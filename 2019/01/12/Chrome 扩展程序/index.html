Chrome 扩展程序

正在使用的一些 Chrome 插件介绍。

每个插件的「点击获取」按钮可以访问 Chrome 网上应用店获取该插件。

服务类

1Password extension

扩展 Mac 或 Windows PC 上的 1Password 应用程序，以便在浏览器中填写和保存密码。

点击获取

印象笔记网页剪藏（Evernote Web Clipper）

使用Evernote扩展程序一键保存精彩网页内容到Evernote帐户。

点击获取

Save to Pocket

Chrome 版 Pocket 扩展件，保存文章、视频等的最佳方式。

点击获取

Todoist：待办事项列表及任务管理器

Todoist 管理着数以百万计的任务，是最好的在线待办事项列表。适用于 Chrome、Web、iPhone、Android、Firefox、Outlook 和其他平台。

点击获取

花瓣网页收藏工具

这个扩展程序可以让你方便地从任意网站上采集喜欢的图片、视频或者网页截图到花瓣网(huaban.com)上。这是花瓣网官方(huaban.com)提供的扩展。

点击获取

基础类

超级拖曳

自定义搜索引擎，在网页选择文本后，按住不放，拖拽即可搜索。

点击获取

Neater Bookmarks 书签管理器

点击弹出书签，文字和间隔大小刚好，带滚动条。这很重要书签一多，想不起来名字的时候。一屏幕显示一部分，想看更多要往下翻半天就很麻烦。支持搜索。

有一个书签侧边栏 ，支持隐藏在侧边，但是容易误触，点击打开的话还要先弹出一个新的标签再来显示书签，长得好看，体验不好。

点击获取

SimpleUndoClose 撤销关闭的标签

这个简单的弹出可让您轻而易举的撤销关闭的标签，支持搜索。

点击获取

Infinity 新标签页(Pro)

在新标签页实现：快速访问网站、自定义搜索、更换壁纸、页面外观自定义设置。

点击获取，或访问 官网 了解更多信息。

标签页管理插件（FreshStart）

大多数人在浏览网页的过程中，都会不知不觉打开了大量的标签页，有时候还不止一个窗口。这些页面，少部分是需要永久保存的，而更多只是目前这个状态下你需要的资料，看完即可关闭。这个 Chrome 的标签管理插件可以暂存所有打开的标签页，并且方便地一键恢复到暂存时的浏览状态。

可给标签页分组命名、可选择性保存标签页、多窗口保存、可单独编辑某分组下的标签、支持云同步。

演示效果：

点击获取，或访问 官网 了解更多信息。

Stylus 网页样式管理器

本来再用 Stylish，但是 Stylish 因涉嫌窃取用户浏览历史，已被 Chrome 和 Firefox 下架。

Stylish 是一款 CSS 编辑器，用户可以使用它为网站创建自定义样式，安装主题和皮肤。它支持多种浏览器，仅 Chrome 浏览器就有超过 186 万用户。
这个插件可以解决 Windows 端 Chrome 字体发虚 的问题。

点击获取

阅读模式

提供与Safari阅读模式功能一致的插件，浏览文章页时候可进入友好的阅读模式，并自定义阅读功能。

点击获取

Anything to QRcode 识别 / 生成二维码

通过右键菜单或地址栏按钮将当前页面地址、选中的文本、链接、图片地址等文本内容生成二维码，也可以解析页面中的二维码图片，完全离线可用，不依赖任何远程文件和服务，内存占用小，界面简洁。
点击获取

右键搜

扩展右键菜单，超过300种功能，方便的选择、组合或自定义页面、划词、图片菜单，并有生成、解析二维码和短网址功能。设置可使用云存储。最新增加功能共享、资源下载、重命名等功能。
点击获取

拷贝为 Markdown

将你在页面中选中的HTML格式转化为Markdown格式。

点击获取

图片类

Imagus（悬停鼠标查看大图插件）

鼠标指针悬停在链接或缩略图上时直接在当前页面的弹出视图上显示这些图片、HTML5 视频/音频和内容专辑。

扩展有众多选项可供设置，要了解扩展的所有功能最好是逐个查阅这些选项。

点击获取

二箱 以图搜图

二箱是一款整合了 Google、百度、必应、TinyEye、SauceNAO 等多个引擎的图片搜索扩展，允许你在一个页面中查看全部相关结果，并按尺寸、相关度等排序。你也可以使用二箱直接截图，或手动上传本地图片搜索，并查看此前的搜索记录。

点击获取

图片助手（ImageAssistant）

当你在 Unsplash、Dribble 和 Pinterest 等网站看到一组美图时，可能会想要将其保存至本地慢慢欣赏，而 图片助手 可以帮助你将其一网打尽。

一款用于嗅探、分析网页图片并提供批量下载等功能,「提取本页图片」允许你按尺寸、类型、URL 等筛选，并一键下载全部图片。

P.S. 如果你曾搜索过 Chrome 图片批量下载扩展，可能会看到不少推荐 Fatkun 的文章。它此前很长一段时间也是我的首选，但在一次更新后出现了 静默弹窗、劫持搜索引擎、后台访问推广网页等行为（你可以在商店评论中找到不少受害者），建议大家移除。

点击获取，或访问 官网 了解更多信息。

View Image

由于被出版商和内容供应商指出侵犯版权，Google 于去年年初移除了图片搜索结果中的「查看图片」和「以图搜图」按钮，带来了许多不便。而 View Image 这款扩展就把它改了回来。

点击获取

捕捉网页截图（Screenshot）

捕捉网页截图，编辑并将它们保存为PDF，JPEG，GIF，PNG或BMP；上传，打印，在Photoshop中打开，复制到剪贴板或电子邮件
点击获取

音频、视频

AHA Music 听歌识曲

用浏览器观看视频时，你的注意力可能会被背景音乐所吸引。你还可以试试 AHA Music这款扩展。

用于网页上的音乐识别，查找和识别您浏览器中播放的任何歌曲。

点击获取，或访问 官网 了解更多信息。

Video Downloader PLUS 视频下载插件

最佳视频下载扩展。 从任何网站下载最流行的媒体类型。
点击获取

翻译类

百度截图翻译插件

百度翻译官方出品，支持中英互译。在浏览器任意界面，使用鼠标截图，即可对图片上的文字进行翻译，双语对照查看。
其调用了百度 OCR 和翻译的相关 API，只需点击图标后拖拽选区，就可以完成从文字识别到成段翻译的全过程。

点击获取 在 Chrome 网上应用店获取百度截图翻译插件

划词翻译

支持谷歌、百度、有道三大翻译和朗读引擎，可以方便的查看、复制和朗读不同引擎的翻译结果。

  • 无需等待，划词后点 [译] 绿色按钮直接弹出翻译。
  • 不会弹出很大个界面框，界面清爽无多余的东西显示。
  • 给出单词基本解释，不会只给一个。
  • 三个平台供选择，弹出的界面右下角可以转到访问翻译的网站。
  • 支持翻译句子。

其他插件：
达达划词翻译：显示一堆英文例句没有取消的选项。
Mate Translate：任何单词都只给一个解释，英文转中文后显示多余的中文拼音。
在线词典助手：不能设置划词后点按钮再翻译，每次划词都会直接弹出翻译界面。
ImTranslator：简单一个词要等联网，网络不好的时候转半天。

对我来说，够用就好。有些单词只是看看，无需加到生词本。
其他更多功能如单词卡制作、收集到单词本、提供例句。可以另外用其他服务本身的软件或者插件。

扇贝助手增强版

任意网页查询并加入生词本、背单词页面简洁、增强词根词缀Webster功能、增加实用快捷键。
经常遇见却不认识的单词，右键点击扇贝助手，查看意思和加入生词本。
可以在选项中设置「查词时默认加入生词库」。

不过这个插件很久没更新了。
